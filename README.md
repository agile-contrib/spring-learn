# spring-learn
Spring学习代码笔记

## Issues
使用的时候遇到任何问题或有好的建议，请点击进入[issue](https://github.com/agile-contrib/spring-learn/issues)！

## 如何使用？

打开```eclipse```后，按照下面顺序导入下面：

```
[File] -> [Import] -> [Maven] -> [Existing Maven Projects]
```

然后选择当前项目即可成功导入。

导入后运行只需要把```Main.java```作为入口，选择```run as Application```即可。
